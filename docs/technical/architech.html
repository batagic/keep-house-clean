v2.0.0 2026-06-10 Static HTML GitHub Pages + VPS No build step

Cấu trúc repo keep-house-clean — frontend HTML ở thư mục gốc, API kho-thoc-api trên VPS Docker + PostgreSQL. Rollback GAS vẫn khả dụng qua config.js. Chi tiết hạ tầng: docs/technical/infrastructure.md.
Quyết định đã lock-in

Các quyết định thiết kế ảnh hưởng trực tiếp đến cấu trúc thư mục.

Backend tách code/kho-thoc-api/ D6 Node.js + Express + PostgreSQL; Docker trên VPS; contract tương thích GAS

Backend — kho-thoc-api

Node.js + Express + pg. Chạy Docker trên VPS, dùng chung eedt-postgres. Contract tương thích GAS.

routes/api.js
GET ping/profiles/logs · POST log, profile, delete_log, redeem
routes/admin.js
POST login · GET/POST passcode generate/revoke
services/profiles.js
CRUD profiles · tạo passcode khi đăng ký bé
services/redeem.js
Validate passcode bcrypt · ghi log REDEEM · trừ Gạo
services/auth.js
JWT admin · bcrypt · rate limit redeem
migrations/
001 profiles+logs · 002 admin+passcodes · 003 profile passcodes

Quy ước quan trọng
1. Thêm trang marketing mới
# Head — load shared CSS theo thứ tự:
assets/css/shared/variables.css → base → nav → layout → buttons → footer → components
assets/css/pages/<ten-trang>.css

# Body — nav đồng bộ:
<nav class="nav" id="mainNav" data-page="ten-trang"></nav>
<script src="assets/js/shared/site-nav.js"></script>
2. Sửa menu toàn site
# Chỉ sửa một file:
assets/js/shared/site-nav.js  →  NAV_ITEMS[]
3. Phân tách runtime vs tài liệu
Thư mụcVai trò
HTML + assets/ (code/ + docs/) Frontend deploy GitHub Pages
code/kho-thoc-api/ Backend — Docker trên VPS
code/deploy/vps/nginx/ Config Nginx — không commit secrets
docs/ Tài liệu — không deploy
ideas/ Prototype — không dùng production
4. Deploy tính năng mới
# Mac:
git push origin main

# VPS (sau mỗi đổi API):
git -C /opt/nhatkyvumua/repo pull
rsync -a repo/code/kho-thoc-api/ code/kho-thoc-api/ --exclude .env
docker compose up -d --build --force-recreate
