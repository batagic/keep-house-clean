Tổng quan hệ thống

Client Layer

HTML/CSS/JS tĩnh trên GitHub Pages (code/). Gọi API qua fetch + header X-Family-Id sau khi mở phiên passcode (tầng 1). Đổi quà: passcode tầng 2 mỗi lần.

API Layer

Node.js Express trong Docker kho-thoc-api, port 3001. Contract tương thích GAS legacy (doGet/doPost).

Data Layer

PostgreSQL kho_thoc trong container eedt-postgres (chung host VPS, DB tách biệt).

Phiên gia đình (Phase 3.1)

unlock_family → X-Family-Id

Client không tự sinh family_id. Ba/mẹ nhập passcode bé → POST type=unlock_family → server trả familyId → lưu localStorage. Ghi nhiệm vụ không hỏi passcode thêm; đổi quà nhập lại mỗi lần. Spec: docs/brd/family-session.md.
