Project Directory Architecture

Kiến trúc Thư mục Dự án

v1.0.0 2026-06-10 Static HTML GitHub Pages No build step

Phản ánh cấu trúc thực tế sau refactor code/ + docs/. Gamification việc nhà cho gia đình — frontend tĩnh, backend hiện tại Google Apps Script (kế hoạch nâng cấp: VPS + PostgreSQL, xem docs/architech.md).
Quyết định đã lock-in

Các quyết định thiết kế ảnh hưởng trực tiếp đến cấu trúc thư mục.

Quyết định ID Ảnh hưởng lên cấu trúc
Tách code/ (frontend) và docs/ (tài liệu) D1 HTML/CSS/JS trong code/; markdown nghiệp vụ trong docs/
Assets tách file — không inline CSS/JS D2 code/assets/css/shared|pages/ + code/assets/js/shared|data|pages/
Nav/footer đồng bộ qua một file JS D3 assets/js/shared/site-nav.js — sửa menu tại một chỗ
Không bắt buộc framework / build tool D4 Deploy thẳng lên GitHub Pages, mở file HTML trực tiếp
Dữ liệu nhiệm vụ & quà tập trung D5 assets/js/data/tasks.js, rewards.js, config.js
— Các quyết định backend / multi-tenant chưa lock-in, sẽ bổ sung dần —

Quy ước quan trọng
1. Thêm trang marketing mới
# Head — load shared CSS theo thứ tự:
assets/css/shared/variables.css → base → nav → layout → buttons → footer → components
assets/css/pages/<ten-trang>.css

# Body — nav đồng bộ:
<nav class="nav" id="mainNav" data-page="ten-trang"></nav>
<script src="assets/js/shared/site-nav.js"></script>
2. Sửa menu toàn site
# Chỉ sửa một file:
assets/js/shared/site-nav.js  →  NAV_ITEMS[]
3. Phân tách docs/ vs code/

Tóm tắt
  • Frontend tĩnh trong code/ — 5 trang HTML + assets/ có tổ chức
  • Tài liệu trong docs/ — nghiệp vụ, kiến trúc, style, datasource
  • Nav/footer: một nguồn site-nav.js — đồng bộ toàn site
  • Dữ liệu tĩnh: tasks.js, rewards.js, config.js
  • Backend hiện tại: Google Apps Script + Sheets — migrate VPS sau
  • Deploy: GitHub Pages — không cần build step
